README Preview
========================

This is a Sublime Text 2 plugin to help you preview already installed other package's README files quickly in you web browser.

## Usage

### Command Palette

* via the command palette (**super+shift+p**) and typing

    * choose Package - preview package's README.
    
## Uses
    
- [GitHub Markdown Rendering API][0]
- [markdown.css][1]
